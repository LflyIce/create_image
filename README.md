# create_image

AI 产品图片生成器 — 基于豆包 seedream 模型的贴膜/贴纸效果图生成工具。

## 功能

- 🎨 **风格图生成**：根据提示词生成各种风格的图案
- 🖼️ **产品贴膜效果图**：将风格图叠加到产品实物图上
- 👤 **用户系统**：手机号注册/登录，点数消耗
- 💰 **充值套餐**：29.9元100点 / 49.9元180点 / 99.9元380点
- 🔧 **管理后台**：用户管理、充值、点数日志、统计面板
- 🌙 **Linear 风格暗色主题**

## 技术栈

- **前端**：React + Ant Design + Vite（TypeScript）
- **后端**：Node.js (server.cjs) + SQLite (better-sqlite3)
- **AI 模型**：豆包 seedream-5-0 / seedream-4-0
- **短信验证**：互亿无线 API
